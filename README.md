# Ejecutar en desarrollo
1. Clonar repositorio
2. Instalar dependencias del proyecto: `npm install`
3. Tener Nest CLI instalado: `npm i -g @nestjs/cli`
4. Levantar la base de datos: `docker-compose up -d`
